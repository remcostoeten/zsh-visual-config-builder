
## Troubleshooting

### Common Issues

1. **Layer not draggable**
   - Check Framer Motion installation
   - Verify drag handlers
   - Check CSS positioning

2. **Type errors**
   - Verify TypeScript version
   - Check type imports
   - Update type definitions

3. **Styling issues**
   - Verify CSS variables
   - Check Tailwind configuration
   - Inspect z-index hierarchy

## Migration Guide

When migrating to a new project:

1. Copy the entire `/modules/layer-system` folder
2. Install required dependencies
3. Add CSS variables to globals.css
4. Import and use components/hooks as needed
5. Test all functionality in isolation
6. Integrate with existing components

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- CSS Variables: IE11 not supported
- Check Framer Motion compatibility
